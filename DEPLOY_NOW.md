# 🚀 지금 바로 배포하기

## ✅ 준비 완료!
- ✅ 프로젝트 빌드 완료 (`docs` 폴더)
- ✅ 배포용 ZIP 파일 생성됨 (`deploy.zip`)
- ✅ UnicornStudio 임베드 포함됨

## 방법 1: Netlify Drop (가장 빠름 - 1분 소요) ⭐

1. **https://app.netlify.com/drop** 접속
2. `deploy.zip` 파일을 드래그 앤 드롭
3. **끝!** 즉시 배포 완료 및 URL 제공

또는 `docs` 폴더를 직접 드래그 앤 드롭해도 됩니다.

## 방법 2: Vercel 웹사이트

1. **https://vercel.com** 접속 및 로그인
2. "Add New Project" 클릭
3. "Deploy" 탭에서 "Browse" 클릭
4. `deploy.zip` 파일 업로드
5. 프로젝트 이름 입력 후 "Deploy" 클릭

## 방법 3: Vercel CLI (터미널에서)

터미널에서 다음 명령어 실행:
```powershell
npx vercel --prod
```

처음 실행 시 브라우저에서 Vercel 계정 인증 필요합니다.

## 방법 4: GitHub + Vercel (자동 배포)

1. GitHub에 코드 업로드
2. Vercel에서 GitHub 레포지토리 연결
3. 자동으로 빌드 및 배포

## 📁 배포 파일 위치

- 빌드된 파일: `porfolio-main/docs/`
- ZIP 파일: `porfolio-main/deploy.zip`

## ✨ 배포 후 확인사항

- Hero 영상이 UnicornStudio 임베드로 표시되는지 확인
- 모든 비디오 파일이 정상 로드되는지 확인
- 반응형 디자인 확인
